Nabi 나비는 X window 환경에서 한글을 입력 가능하게 하는 X Input Method 입니다.
기본적으로 GTK+2를 사용하고 있고 System Tray에 들어갈수 있도록 만들어 졌습니다.

관심 있는 분들의 참여 바랍니다.

설치

컴파일해서 설치하는 방법은 일반적인 프로그램들 처럼 

./configure
make
make install
하시면 됩니다.

사용방법

한영 전환은 "한/영"키를 누르거나 Shift-Space를 누르면 됩니다.
자판 선택은 시스템 트레이의 상태정보 아이콘에서 오른쪽 버튼을 누르면 나오는 메뉴에서 선택을 하면 됩니다.
