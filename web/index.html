나비는 사용하기 쉬우면서도 강력한 기능을 가진 GTK+2 로 만들어진 GNU 한글 XIM 프로그램입니다. X 에서 한글을 사용하기 위하연 자유롭게 사용 배포할수 있습니다. 현재 많은 시스템과 플렛폼에서 작동을 하며, 한글을 사용하기 위한 기본 입력기로 사용되고 있습니다.

홈페이지 개편 2004-10-02 

작게 나마 홈페이지 디자인을 바꾸었습니다. 또한 영문 나비 페이지또한 준비 중입니다.

나비 0.14 버전 출시 2004-08-28 

설정창 제공, java 프로그램에서 한글 입력을 더 자연스럽게 수정, 한자/심벌 입력리스트에서 갯수를 9개로 수정, 지원하지 않는 locale의 경우 워닝 출력, en_US.UTF-8 locale에서도 한글 입력 지원

나비 0.13 버전 출시 2004-06-10 

세션기능 지원, 한자 입력시 뜻도 보여줌, 메모리 릭 문제 등 자잘한 버그 수정
